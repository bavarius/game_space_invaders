# Space Invaders Clone
This is a clone of the famous Arcade game.
It was programmed in Python using **Turtle graphics**.

## Game Control
| Key | Action |
|-----|-----|
| Arrow Left | Move ship left |
| Arrow Right | Move ship right |
| Space | Fire |
